# Smart Data Analyzer

## Description

This project analyzes a list of numbers entered by the user and provides useful statistics.

This project was built as part of my learning journey in Python.

## Features

* Sum of numbers
* Average value
* Maximum and Minimum
* Count of elements
* Even and Odd separation

## Technologies Used

* Python

## How to Run

### Using Python

python main.py

### Using Jupyter Notebook

Open `Data_Analyzer.ipynb` and run all cells.

## Example

Input:
10 15 20 56

Output:
Sum: 101
Average: 25.25
Max: 56
Min: 10
Count: 4
Even: [10, 20, 56]
Odd: [15]
